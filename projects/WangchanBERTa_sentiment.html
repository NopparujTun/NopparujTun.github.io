Fine-tuning WangchanBERTa for Sentiment Analysis¶

Introduction¶

งานวิเคราะห์ความรู้สึก (sentiment analysis) คือการให้คอมพิวเตอร์อ่านข้อความแล้วบอกว่าผู้เขียนรู้สึกอย่างไร ในโปรเจกต์นี้เราจะจำแนกข้อความภาษาไทยออกเป็น 3 ความรู้สึก: บวก (positive), กลาง (neutral), และลบ (negative) เป้าหมายหลักคือการ เปรียบเทียบสองแนวทาง ที่ต่างกันคนละยุค เพื่อดูว่าความซับซ้อนที่เพิ่มขึ้นนั้นคุ้มค่าแค่ไหน:

  1. Baseline — TF-IDF + Naive Bayes เป็นวิธีคลาสสิกของ NLP ที่มองข้อความเป็นเพียง "ถุงคำ" (bag-of-words) นับความถี่ของคำแล้วจำแนกด้วยความน่าจะเป็น จุดเด่นคือเร็ว เข้าใจง่าย และไม่ต้องใช้ GPU แต่ข้อจำกัดคือมันไม่เข้าใจบริบทหรือลำดับของคำ เช่น แยกการประชด

  2. WangchanBERTa (fine-tuning BERT) เป็นโมเดลภาษาสมัยใหม่แบบ Transformer ที่ถูก pre-train มาแล้วบนข้อความภาษาไทยจำนวนมหาศาล ทำให้มันเข้าใจความหมายเชิงบริบทได้ เราเพียงนำมา fine-tune ต่อกับงานของเรา

Dataset: ใช้ Wisesight Sentiment ซึ่งเป็นชุดข้อมูลรีวิว/ความคิดเห็นภาษาไทยจริงที่มีการ label แล้ว และแบ่ง train / validation / test มาให้เรียบร้อย

Hyperparameter Tuning: จูนค่า alpha ของ Naive Bayes¶

In [10]:
from sklearn.naive_bayes import MultinomialNB

best_alpha, best_f1 = None, -1
for alpha in [0.1, 0.5, 1.0, 2.0]:
    m = MultinomialNB(alpha=alpha).fit(X_train, y_train)
    f1 = f1_score(y_val, m.predict(X_val), average="macro")
    print(f"alpha={alpha}: val F1={f1:.4f}")
    if f1 > best_f1:
        best_alpha, best_f1 = alpha, f1

print("best alpha:", best_alpha)

from transformers import AutoTokenizer

MODEL_NAME = "airesearch/wangchanberta-base-att-spm-uncased"

tokenizer = AutoTokenizer.from_pretrained(MODEL_NAME)

def tokenize(batch):
    tokenized = tokenizer(
        batch["texts"],
        truncation=True,
        max_length=128
    )
    tokenized["labels"] = batch["category"]
    return tokenized


train_token = train_dataset.map(tokenize, batched=True, batch_size=len(train_dataset))
test_token = test_dataset.map(tokenize, batched=True, batch_size=len(test_dataset))
val_token = val_dataset.map(tokenize, batched=True, batch_size=len(val_dataset))

def compute_metrics(p):
  pred = np.argmax(p.predictions, axis=1)
  return {"accuracy": accuracy_score(p.label_ids, pred), "f1": f1_score(p.label_ids, pred, average="macro")}

trainer = Trainer(
    model=model,
    args=args,
    train_dataset=train_token,
    eval_dataset=val_token,
    data_collator=data_collator,
    compute_metrics=compute_metrics,
)
